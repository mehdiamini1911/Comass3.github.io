Compass 3™ Interface - Stereographic Analysis

Introduction

Compass 3™ is a sophisticated software package designed for stereonet-photogrammetry-based rock mass characterization and kinematic analysis of rock slopes. Developed by Mehdi Amini, this professional tool enables geotechnical engineers and rock mechanics professionals to efficiently analyze rock slope stability through comprehensive discontinuity analysis.

Primary Functions

Data Import

Import joint orientation data from various formats

Visualization

Visualize discontinuities on stereonets

Slope Definition

Define slope face and excavation geometries

Failure Analysis

Perform kinematic failure mode analysis

About the Developer

Senior Rock Mechanics Specialist

Mehdi Amini, Ph.D., P.Eng., MBA, is a senior Rock Mechanics specialist with over 23 years of experience in civil and mining engineering projects. Currently an Associate at Thurber Engineering Ltd., Mehdi combines deep technical expertise with practical field experience to solve complex rock mechanics challenges.

He is the developer of Compass 3™, a specialized software designed to assist engineers in rock slope analysis. Compass 3™ integrates advanced engineering principles with user-friendly tools for slope stability assessment, 3D visualization, kinematic analysis, and data management, helping professionals make informed decisions efficiently and accurately.

Driven by a passion for innovation and precision in geotechnical engineering, Mehdi continues to enhance Compass 3™, ensuring it meets the evolving needs of engineers and researchers in the field.

Key Features

Joint Data Management

  • Import from CSV or OBJ files
  • Automatic data validation
  • Statistical analysis of discontinuity sets
  • Clustering and set identification

Stereonet Analysis

  • Equal area projections
  • Contouring of pole densities
  • Mean set orientation calculation
  • Friction cone visualization

Kinematic Analysis

  • Planar failure analysis
  • Wedge failure analysis
  • Toppling failure analysis

Slope Geometry

  • 3D visualization
  • Import LiDAR or OBJ files
  • High-resolution photogrammetry display
  • Definition of multiple slope faces
